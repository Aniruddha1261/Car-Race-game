# Car-Race-game

## Description:
- A simple car racing game where you try to make maximum score.
- How to Play   
    - Use LEFT/Right Arrow key to controll the car.
    - Goal is to avoid collision with other vehicles and make maximum score.
### Tech Stack
- HTML
- CSS 
- JavaScript
